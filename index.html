What is Docker

Docker is a tool designed to make it easier to create, deploy and run applications by using containers. The container allow a developer to package up an application with all of the parts it needs, such as libraries and other dependencies and ship it all out as one package. Docker is developed primarily for Linux, it allows independent containers to run within a single Linux instance, avoiding the overhead of starting and maintaining a virtual machine. A virtual machine can run several containers simultaneously.

https://opensource.com/resources/what-docker

What is Nginx

NGINX is an open source software for a web server which can also be used as a reverse proxy, mail proxy, load balancer, and HTTP cache server. The software was created by Igor Sysoev and first publicly released in 2004. Nginx web server is one of the leading web servers used worldwide behind the Apache Web server and Microsoft's IIS.

What is VM (virtual machine)

VM - virtual machine is an emulation of a computer system. The virtual machine is based on computer architect, comprised of a set of specification and configuration files and is backed by the physical resources of a host. Every virtual machine has virtual devices that provide the same functionality as physical hardware.
